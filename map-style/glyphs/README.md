# Glyphs

Dieses Verzeichnis dient der lokalen Ablage von MapLibre-kompatiblen Schriftarten im Protocol Buffer Format (PBF). Dieses Verzeichnis ist derzeit ein vorbereiteter Ablageort (Scaffold); produktiv nutzbare Artefakte sind noch nicht vollständig eingecheckt bzw. generiert.
Die MapLibre `style.json` erwartet die Schrift "Noto Sans Regular" und ist auf den relativen Pfad `./glyphs/{fontstack}/{range}.pbf` konfiguriert.
Die konkreten PBF-Dateien fehlen noch (siehe `map-style/ASSETS.md`).
