# Glyphs

Dieses Verzeichnis ist als lokaler Ablageort (Scaffold) für MapLibre-kompatible Schriftarten im Protocol Buffer Format (PBF) vorbereitet. Produktiv nutzbare Artefakte sind noch nicht vollständig eingecheckt bzw. generiert.
Die MapLibre `style.json` erwartet die Schrift "Noto Sans Regular" und ist auf den relativen Pfad `./glyphs/{fontstack}/{range}.pbf` konfiguriert.
Die konkreten PBF-Dateien fehlen noch (siehe `map-style/ASSETS.md`).
