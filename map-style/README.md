# Weltgewebe Map Style

This directory serves as the single source of truth for the sovereign visual presentation of the Weltgewebe basemap, as outlined in the `docs/blueprints/map-blaupause.md`.

## Purpose

To ensure total visual sovereignty and compliance (no silent vendor dependencies), the entire map style definition lives here. The `style.json` is designed to consume PMTiles artifacts generated by our reproducible pipeline.

## Runtime Assumptions (Important)

* glyphs are referenced via relative paths and must be served alongside the `style.json`
* `pmtiles://` URLs require a protocol handler in the MapLibre runtime (not part of this configuration)
* The current configuration is a structural artifact, not a fully wired runtime setup

## Structure

* `style.json`: The standard MapLibre style document.
* `colors.json`: High-level color definitions and palette.
* `glyphs/`: Local, self-hosted font stacks in PBF format to prevent external font service dependencies. (Fetched via `scripts/basemap/fetch-glyphs.sh`)

> **Note on Sprites:** To maintain a visually calmed infrastructure basemap, sprites (icons/patterns) are intentionally omitted from this map style MVP. Visual semantics belong in the Weltgewebe overlay.
