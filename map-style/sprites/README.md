# Sprites

Dieses Verzeichnis ist als lokaler Ablageort (Scaffold) für MapLibre-kompatible Spritesheets (`sprite.png`,
`sprite.json`, `sprite@2x.png`, `sprite@2x.json`) vorbereitet. Produktiv nutzbare Artefakte sind noch nicht vollständig
eingecheckt bzw. generiert. Die MapLibre `style.json` ist auf den relativen Pfad `./sprites/sprite` konfiguriert. Die
konkreten Assets fehlen noch (siehe `map-style/ASSETS.md`).
